# core-native-sdl

 * a native core for <https://github.com/gamelly/gly-engine>

## How to build

```bash
git clone https://github.com/gamelly/core-native-sdl
```

```bash
cmake -Bbuild -H.
```

```bash
make -C build
```

## How to run

```bash
./build/bin/game -g vendor/engine/examples/asteroids/game.lua
```
